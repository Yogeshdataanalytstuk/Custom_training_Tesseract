Data(any picture format)  with annotation(.box format) .pBox file should be named as same of picture file(example eng.aerail.exp0.jpg is the picture format and box file should be ng.aerail.exp0.box) 
Box file should be formated as Single line for each character annotation(format: A 1 2 3 4 0)
- A ia the annotation character
- 1 is the Xmin value
- 2 is the Ymin value
- 3 is the Xmax value
- 4 is the Ymax value
- 0 page of annotation(if it has more than 1 first page is 0 and it continue on)
